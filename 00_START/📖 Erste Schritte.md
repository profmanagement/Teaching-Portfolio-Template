# 🚀 Erste Schritte – Lies mich zuerst!

Willkommen in deinem Lehrportfolio-Vault. Diese Anleitung erklärt, wie alles zusammenhängt und wie du anfängst.

---

## Wie dieser Vault strukturiert ist

```
Teaching-Portfolio/
│
├── 00_START/              ← Du bist hier. Dashboard & Einstieg.
├── 01_Lehrphilosophie/    ← Dein pädagogisches Kernstück (mit Versionsarchiv)
├── 02_Lehrveranstaltungen/← Alle Kurse, Syllabi, Notizen
├── 03_Reflexionen/        ← Laufendes Lehr-Journal
├── 04_Feedback/           ← Studierenden-, Peer-, Selbstfeedback
├── 05_Entwicklung/        ← Ziele, Weiterbildungen, Meilensteine
├── 06_Materialien/        ← Ausgewählte Lehrproben & Innovationen
├── 07_Portfolio-Ausgabe/  ← Kuratierte externe Version
├── 08_Lehrdesigns/        ← Didaktische Methoden-Bibliothek
├── 09_Archiv/             ← Abgeschlossene Semester
└── Templates/             ← Wiederverwendbare Vorlagen
```

---

## Empfohlener Einstieg (Schritt für Schritt)

### Woche 1: Fundament legen
- [ ] Öffne [[01_Lehrphilosophie/Lehrphilosophie]] und beantworte die erste Frage: *„Gute Lehre bedeutet für mich..."*
- [ ] Lege für jede aktuelle Lehrveranstaltung eine Notiz an (Vorlage: [[Templates/Template – Lehrveranstaltung]])
- [ ] Starte deinen [[03_Reflexionen/Reflexions-Log]] mit einem ersten Eintrag

### Woche 2–4: Routine aufbauen
- [ ] Schreibe nach jeder dritten Lehrveranstaltung eine kurze Reflexion (5–10 Min.)
- [ ] Trage Weiterbildungen und Lektüre in [[05_Entwicklung/Entwicklungsplan]] ein
- [ ] Verknüpfe Notizen miteinander (z. B. Reflexion → Lehrphilosophie)

### Am Semesterende: Auswerten & Archivieren
- [ ] Werte das Feedback aus: [[Templates/Template – Feedback-Auswertung]]
- [ ] Aktualisiere die Lehrveranstaltungsnotiz mit Reflexion & Learnings
- [ ] Aktualisiere deine Lehrphilosophie falls nötig
- [ ] Snapshot der Lehrphilosophie in [[01_Lehrphilosophie/Versionen/CHANGELOG]] eintragen
- [ ] Destilliere Highlights in [[07_Portfolio-Ausgabe/Präsentationsportfolio]]
- [ ] Abgeschlossene Semester-Notizen nach [[09_Archiv/README|09_Archiv/]] verschieben

### Laufend: Methoden dokumentieren
- [ ] Genutzte Methoden in [[08_Lehrdesigns/Übersicht Lehrdesigns]] vermerken
- [ ] Eigene Erfahrungen in die jeweilige Methoden-Notiz eintragen (Abschnitt „Meine Erfahrungen")
- [ ] Neue Methoden mit [[Templates/Template – Lehrdesign]] anlegen

## Obsidian-Tipps für diesen Vault

| Funktion | Shortcut | Wozu |
|---|---|---|
| Graph-Ansicht | `Strg/Cmd + G` | Wissensvernetzung sichtbar machen |
| Schnellsuche | `Strg/Cmd + O` | Notizen öffnen |
| Neue Notiz | `Strg/Cmd + N` | Neue Idee festhalten |
| Template einfügen | `Alt + E` (nach Plugin-Setup) | Vorlage nutzen |
| Backlinks | Seitenleiste rechts | Sehen, was womit verknüpft ist |

### Empfohlene Plugins (Community)
- **Templater** – Für dynamische Vorlagen mit Datum etc.
- **Dataview** – Für automatische Tabellen (im Dashboard genutzt)
- **Calendar** – Für tagesbasierte Journaleinträge
- **Periodic Notes** – Für wöchentliche/monatliche Reflexionsroutinen

---

## Die drei goldenen Regeln dieses Portfolios

> **1. Verknüpfen statt stapeln.**
> Jede neue Notiz: Wo gehört sie hin? Was verbindet sie mit Bestehendem?

> **2. Reflektieren statt nur dokumentieren.**
> Eine Notiz ohne Interpretation ist eine Datei. Erst die Deutung macht sie wertvoll.

> **3. Kuratieren statt alles sammeln.**
> Das Portfolio zeigt, wer du als Lehrperson bist – nicht alles, was du je gemacht hast.
